== Description ==
The Fart Stories Addon allows users to create, share, and categorize story-related stories. Integrated with Fart Calculator, it provides a fun platform for users to submit and share their most memorable story experiences.

== Installation ==
1. Make sure you have the Fart Calculator plugin installed and activated.
2. Upload the Fart Stories Addon to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
